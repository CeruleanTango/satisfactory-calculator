# Satisfactory Production Calculator

A containerized production chain calculator for the factory building game 'Satisfactory'

## Goals ##
Practical learning experience for:
- Docker multi-container applications
- IaC using Terraform
- Database design and management
- API development

## Features ##
- Production chain calculation: Input desired output rate, get required resource values
- Raw material tracking: Calculate exact resource extraction needs
- Building requirements: Determine how many buildings are needed for intermediate parts and final parts for desired output rate
- Power consumption: Total required power needs for factory

## Tech Stack ##
- **Backend** Python 3.11, FastAPI
- **Database** PostgreSQL 15
- **Containerization** Docker, Docker Compose
- **Infrastructure** Terraform

## Prerequisites ##
- Docker Desktop (Windows)
- Docker Compose
- Git
